Rumah saya
Rumah saya ada di kota Gotemba,Shizuoka. Kota Gotemba terkenal dengan gunung Fuji dan Outlet mal yang paling besar di Jepang. Ruma saya dua lantai dan kamar saya ada di lantai dua. Di kamar saya, tidak ada banyak mebel. Hanya ada tempat tidur dan lemari kecil di Kamar saya. Tidak ada meja karena saya tidak belajar di kamar saya. Alih-alih, saya belajar di ruang keluarga. Jika saya terdengar suara hidup, saya bisa berkonsentrasi dengan lebih baik. Jadi saya biasanya pergi ke kamar saya ketika saya tidur. Saya sangat suka tidur ,jadi saya meminta orang tua saya untuk membeli ranjang ganda. Itu mungkin terlalu besar untuk saya, tetapi saya bisa tidur dengan nyaman.Waktu saya mau berbaring-baring, saya menghabiskan di ruang keluarga. Keluarga saya juga suka menghabuskan di sini karena kami bisa menonton televisi besar dan kandang anjing ada di situ.Lebih-lebih lagi, Ibu saya menbeli bantal besar bernama “Yogibo” yang kami bisa duduk dan berbaring tahun lalu. Kami sering memperebutkan siapa yang mempergunakan itu. Kamar kesukaan saya kamar mandi. Kamar mandi biasanya adadi lantai satu, tetapi kamar mandi kita ada di lantai dua. Pasalnya,ayah saya mau merancangkan seperti itu. Itu ada rasa keterbukaan dan cahaya masuk dari jendela siang hari. Di dekat kamar mandi, serambi ada dan angin saya sangat suka pergi ke situ. Ibu saya membawa pergi anjin saya ke situ setiap hari.
