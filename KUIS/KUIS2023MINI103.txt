Saya tinggal di apartmen di Provinsi Edogawa.
Dari stasiun ke apartmen saya memakan waktu kira-kira 5(lima) menit dengan berjalan kaki. Apartmen tempat saya tinggal bersih karena konstruksi kira-kira tujuh tahun.Saya tinggal di sana selama tujuh. 
Di sebelah utara apartmen saya ada supermarket, toko obat dan restoran keluarga.
Di sebelah selatan apartmen saya ada stasiun, SD dan SMP. Saya pergi ke SMP itu saja. Pada sore, banyak murid-murid berjalan.
Di sebelah timur apartmen saya minimarket. Ada "LAWSON", "Family mart" dan "Seven-Eleven".
Di sebelah barat apartmen saya ada AEON. Itu kecil tetapi bersih.
Tidak ada pusat perbelanjaan besar. Lalu, waktu saya mau berbelanja, saya pergi ke pusat berbelanjaan memakan waktu kira-kira 15(lima-belas) menit dengan kereta.
Saya tidak ada rencana pindah tetapi saya mau hidup seorang di sekitar sekolah.
