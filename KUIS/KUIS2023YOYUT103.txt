Saya tinggal di apartemen mewah di Kozunomori.
Dari stasiun ke apartemen mewah saya memakan waktu kira-kira lima menit dengan berjalan kaki.
Bangunan apartemen mewah yang saya tinggali belum lama, tetapi tidak baru.
Saya sudah tinggal di sana selama dua belas tahun.
Sekitar apartemen mewah saya ramai.
Di sebelah utara dan selatan apartemen mewah saya ada dua apartemen mewah.
Di sebelah timur apartemen mewah saya ada pusat perbelanjaan.
Di dalam pusat perbelanjaan itu ada toko buku, game center tempat saya bekerja, supermarket, dan lain-lain.
Ibu saya bekerja di sana juga.
Di sebelah utara apartemen mewah saya ada stasiun dan universitas.
Banyak orang memanfaatkan stasiun itu.
Di sebelah barat apartemen mewah saya ada taman.
Pada waktu saya masih kecil, saya bermain ke taman itu dengan teman saya.
Di sebelah selatan apartemen mewah saya ada SD.
Pada waktu saya murid SD, saya ke sana.
Di barat daya apartemen mewah saya ada perpustakaan.
Perpustakaan itu tidak besar, tetapi baru dan bersih.
