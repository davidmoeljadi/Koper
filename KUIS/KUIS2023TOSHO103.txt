Sekitar rumah/apartement saya

Saya tinggal di apartement mewah di Chiba. Dari stasiun ke apatement mewah saya memakan waktu kira-kira 4 menit dengan berjalan kaki. Apartement mewah tempat saya tinggal tidak baru, tetapi aman dan nyaman. Saya tingggal di sana selama 16 tahun.

Sekitar apartement mewah saya tidak ramai. Di depan apartement mewah saya jalan di belakangnya ada toko dokter gigi. Di sebelah kanan apartement mewah saya ada toko daging. Di sebelah utara terdapat stasiun NishiChiba, seiyu, Universitas Chiba, dan sukiya. Saya suka makan malam dengan teman di sana. Kadang-kadang saya berlari di sekeliling Universitas Chiba. Di sebelah timur terdapat stasiun Chiba, PERIE, SOGO, YODOBASHICAMERA, bioskop, dan dsb. Saya sukan berbelanja di sana. Ada banyak toko baju di stasiun Chiba. Tetapi toko yang saya suka tidak ada di sana. Jadi saya berbelanja di Lalaport Minamifunabashi.  

Saya senang sekali tinggal di apartement mewah itu karena saya tinggal di sana waktu lama dan ada banyak teman-teman saya. Orang tetangga juga baik hati.

Tetapi saya mau pindah rumah karena saya mau hidup sendiri.
